### What is it?

Black Screen is a modern terminal emulator based on [Electron](https://github.com/atom/electron) and exploiting
capabilities of contemporary interfaces in contrast to most of the terminal emulators.

### Technologies

* TypeScript
* Electron
* NodeJS
* ReactJS
* SASS

### Installation

```bash
npm run install-all
```

### Run

```bash
npm run forest-run
```
